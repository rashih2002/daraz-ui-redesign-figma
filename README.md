# 🛒 Daraz Mobile App – UI/UX Redesign

A UI/UX redesign concept of the Daraz mobile application created as a university group project using Figma. The project focused on improving user experience, visual consistency, navigation, and accessibility through modern mobile interface design principles.

---

## 📌 Project Overview

- **Project Type:** University Group Project
- **Design Tool:** Figma
- **Platform:** Mobile Application
- **Focus:** UI/UX Design

---

## 🎯 Objectives

- Improve user experience and usability
- Create a cleaner and more intuitive interface
- Enhance visual consistency across screens
- Apply user-centered design principles
- Improve accessibility and navigation

---

## 👥 Team Information

- **Project Type:** Group Assignment
- **My Role:** UI/UX Design Contributor

### My Contributions

- User flow planning

- Prototype collaboration
